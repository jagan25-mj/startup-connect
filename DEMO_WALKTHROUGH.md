# CollabHub Demo Walkthrough

A 5-minute judge-ready demo flow for CollabHub platform.

## Demo Scenario

We'll demonstrate CollabHub's core value: connecting founders with talent through AI-assisted matching.

**Time**: 5 minutes  
**Participants**: Founder, Talent, Investor  
**Outcome**: Complete workflow with real-time collaboration features

## Pre-Demo Setup

### Preparation (Do this before showing)

1. **Create test accounts** (or use pre-made accounts):
   - Account 1: Founder (email: founder@collabhub.demo)
   - Account 2: Talent (email: talent@collabhub.demo)
   - Account 3: Investor (email: investor@collabhub.demo)

2. **Open 3 browser windows** (or 3 tabs in incognito):
   - Window 1: Logged in as Founder
   - Window 2: Logged in as Talent
   - Window 3: Logged in as Investor

3. **Pre-create one startup** as founder for demo
